# Eclipse代码自动换行快捷键
### 功能描述
&emsp;&emsp;首先说明一下，这里说的自动换行指的是eclipse编辑器中某一行文字太长时换行展示，而不是真正的将一行拆分成多行。
### 如何使用
&emsp;&emsp;Eclipse 从Neon版本开始就加入了“Word wrap in text editors”，并且有快捷键 **Alt+Shift+Y**
